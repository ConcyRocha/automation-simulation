Domótica:

Utilizando sistemas inteligentes ou integrados em uma controladora de automação, é possível monitorar e gerenciar toda a residência através de uma central via smartphone, tablet e internet. Para a criação de um sistema inteligente é conveniente estudar sobre ferramentas, como Arduíno e Raspberry pi, que são hardwares de baixo custo e acessíveis tanto para profissionais em programação e eletrônica quanto para curiosos que gostam do tema. Mas afinal, o que é um Arduíno e um Raspberry pi? E quais suas funcionalidades?



Segundo o cientista da computação Adilson Thomsen, Arduino, Figura 1, é uma a placa composta por um microcontrolador, circuitos de entrada/saída e que pode ser facilmente conectada à um computador e programada via IDE (Integrated Development Environment, ou Ambiente de Desenvolvimento Integrado) utilizando uma linguagem baseada em C. Já o Raspberry pi, Figura 2, é um minicomputador, semelhante ao PC (personal computer). A diferença é que este dispositivo é compacto e possui todos os principais componentes de um computador numa pequena placa do tamanho de um cartão de crédito. Este tipo de hardware, assim como o Arduino, pode ser utilizado para automatização de tarefas através de programação via lDE. Porém, antes de continuar o assunto sobre como essa programação é feita, necessita-se saber mais sobre o que é programação e quais as linguagens, geralmente, usadas para estes fins.

O que é programação?

A programação é um processo de escrita, testes e manutenção de programas de computadores. Esses programas, por sua vez, são compostos por conjuntos de instruções determinados pelo programador que descrevem tarefas a serem realizadas pela máquina e atendem diversas finalidades. As instruções do programador para a máquina são feitas a partir de linguagens de programação.

Quais são as linguagens de programação e suas aplicações no sistema?


As principais linguagens de programação aplicadas para os hardwares de controle são as linguagens C, Python e Java. Para fazer a conexão via servidor web são utilizadas as linguagens Javascripts, Java, PHP e Nodejs.
