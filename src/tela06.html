Tipos de hardware de controle: placas de desenvolvimento para domótica


Utilizando sistemas inteligentes ou integrados em uma controladora de automação, é possível monitorar e gerenciar toda a residência através de uma central via smartphone, tablet e internet. Para a criação de um sistema inteligente é conveniente estudar sobre ferramentas, como Arduíno e Raspberry Pi, que são hardwares de baixo custo e acessíveis tanto para profissionais em programação e eletrônica quanto para curiosos que gostam do tema. Mas afinal, o que é um Arduíno e um Raspberry pi? E quais suas funcionalidades?



O Arduino, mostrado na Figura 4, é uma a placa composta por um microcontrolador, circuitos de entrada/saída e que pode ser facilmente conectada à um computador e programada via IDE (Integrated Development Environment, ou Ambiente de Desenvolvimento Integrado) utilizando uma linguagem baseada em C. Já o Raspberry Pi, mostrado na Figura 5, é um minicomputador, semelhante ao PC (personal computer). A diferença é que este dispositivo é compacto e possui todos os principais componentes de um computador numa pequena placa do tamanho de um cartão de crédito. Este tipo de hardware, assim como o Arduino, pode ser utilizado para automatização de tarefas através de programação utilizando uma linguagem de computador.

O que é programação?

A programação é um processo de escrita, testes e manutenção de programas de computadores. Esses programas, por sua vez, são compostos por conjuntos de instruções determinados pelo programador que descrevem tarefas a serem realizadas pela máquina e atendem diversas finalidades. As instruções do programador para a máquina são feitas a partir de linguagens de programação.

Quais são as linguagens de programação e suas aplicações no sistema?


As principais linguagens de programação aplicadas para os hardwares de controle, como o Arduino e o Rapberry PI, são as linguagens C, Python e/ou Java. Para desenvolver uma aplicação web que permite o controle remoto dos hardwares, são utilizadas linguagens como Javascripts, Java, PHP e/ou NodeJs.
